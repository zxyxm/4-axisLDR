# 电机说明：
28步进，步距角1.8度
# 步进电机驱动说明：
驱动器已经设置为3200细分，一圈就是3200脉冲
脉冲当量为指定每单位发送的脉冲数，支持5位小数精度。
一般设为电机走1mm或1°的所需脉冲数。

假设电机U=3200脉冲转一圈，那么电机转1°对应的UNITS：U/360=3200/360，
此时MOVE(1)指令的含义为电机转1°。
此时MOVE(180)指令的含义为电机转180度。
